Как зашифровать системный диск с помощью BitLocker

27.03.2017 | 10.06.2017 |
125 | 1

Это быстрая заметка-напоминание для меня, если, вдруг, в будущем понадобиться быстро найти информацию по шифрованию системного раздела с помощью BitLocker.

Если ОС не разрешает включить BitLocker (так было у меня на Windows 10, версия enterprise) — необходимо с помощью комбинации клавиш Win + R вызвать окно «Выполнить» и в нём прописать gpedit.msc (команда вызовет редактор локальной групповой политики).

Затем перейдите по пути Политика “Локальный компьютер” Административные шаблоны Компоненты Windows Шифрование диска BitLocker Диски операционной системы дважды кликните по параметру Настроить требование дополнительной проверки подлинности при запуске выберите «Включено»:

После этого можно перейти в панель управления «Шифрование диска BitLocker» и включить BitLocker.
